# AI-Powered News Analysis Tool (Single-Pass Evaluation)

## Overview
This project implements a lightweight AI-based system to analyze news content and assess credibility using a **single-pass LLM evaluation** approach.

The goal is to quickly identify potentially misleading or unreliable content without complex multi-stage pipelines.

## Key Features
- One-pass LLM-based analysis
- Content reasoning and explanation
- Fast inference workflow

## Tech Stack
- Python
- LLM API
- NLP techniques
- Google Colab

## How It Works
1. User provides a news article or text
2. LLM analyzes content for logical consistency and red flags
3. Outputs a credibility-oriented assessment with explanation

## Use Cases
- Fake news detection (basic)
- Content moderation assistance
- Media literacy tools
